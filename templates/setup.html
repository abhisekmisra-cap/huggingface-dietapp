üÜì It's Completely Free!

Hugging Face API tokens are free and give you access to powerful AI models without downloading gigabytes of data. The process takes less than 2 minutes!

üìù Step-by-Step Instructions

Step 1: Create Account

Go to huggingface.co/join and create a free account.

What you need: Just an email address - no credit card required!

Step 2: Generate Token
  1. Visit huggingface.co/settings/tokens
  2. Click "New token"
  3. Give it a name like "Diet Plan Generator"
  4. Select "Read" role (sufficient for this app)
  5. Click "Generate a token"
  6. Copy the generated token
Step 3: Use Your Token

Option 1: Web Interface (This Page)

Simply paste your token in the "API Token" field when generating a diet plan.

Option 2: Set Environment Variable

Windows (Command Prompt):
set HUGGINGFACE_API_TOKEN=your_token_here
Windows (PowerShell):
$env:HUGGINGFACE_API_TOKEN='your_token_here'
Linux/Mac:
export HUGGINGFACE_API_TOKEN=your_token_here

Option 3: Command Line

python main.py --token your_token_here --age 30 --weight 70 --nationality Indian

üöÄ Quick Test

Want to test your setup without the web interface?
